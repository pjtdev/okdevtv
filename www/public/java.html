JDK (Java Development Kit)

  • http://java.sun.com/ 위치에서 Java SE 다운로드
  • 기본 설치(옵션 바꾸지 않고 Next, Next로 설치 완료)
  • 64비트 윈도우에 32비트 jdk 설치 가능 C:\Program Files (x86) 하위에 설치

JDK 설치

  • 기본 설정으로 설치
  • JDK 설치 후 JRE(Java Runtime Environment) 설치
  • JRE는 javac(컴파일러)가 제외된 브라우저 애플릿 등의 실행환경

JAVA_HOME 환경변수

  • JAVA_HOME 환경변수는 자바 개발의 시작
  • 윈도우에서 환경변수를 설정하는 방법
  • 탐색기에서 컴퓨터를 선택하고 컨텍스트 메뉴의 속성을 열기
  • 좌측에서 고급 시스템 설정을 클릭
  • 고급 탭에서 하단의 환경 변수(N)... 버튼을 클릭
  • 시스템 변수 > 새로 만들기 버튼 클릭. 변수 이름에 JAVA_HOME을 입력하고, 변수 값에 설치된 경로를 붙여넣기
  • 시스템 변수 Path 항목 열어서 맨 앞에 %JAVA_HOME%\bin; 을 추가하고, 모든 창 닫기

IDE(Integrated Development Environment)

  • 통합 개발 환경은 코딩+컴파일+실행/디버깅이 가능한 도구
  • 자바는 이클립스가 대세
  • 인텔리J는 자바개발시 무료, JavaEE 개발용은 유료
